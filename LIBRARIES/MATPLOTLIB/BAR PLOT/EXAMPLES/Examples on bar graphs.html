#Problem-1
#In a firm of 400 employees, the percentage of monthly salary saved by each employee is given in the following table.
#Represent it through a bar graph.
#--------------------------------------------------------------------------------------------------------------------------
#		Savings (in percentage)	Number of Employees(Frequency)
#--------------------------------------------------------------------------------------------------------------------------
#			20					105
#			30					199
#			40					29
#			50					73
#--------------------------------------------------------------------------------------------------------------------------
In[4]:
#Draw the Bar graph the above problem
import numpy as np
from matplotlib import pyplot as plt
Savings=np.array([20,30,40,50])
NOEmp=np.array([105,199,29,73])
plt.bar(Savings,NOEmp)
plt.show()
